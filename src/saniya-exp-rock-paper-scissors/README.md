# Rock, Paper, Scissors Game

A simple Python game where you can play **Rock, Paper, Scissors** against the computer.  

---

## Features

- Play multiple rounds against the computer.
- Enter your choice: `rock`, `paper`, or `scissors`.
- Computer randomly selects a move.
- Shows the winner or if itâ€™s a tie.
- Option to quit the game anytime.

---

## How to Play

```bash
1. Run the Python script:
   python main.py

2. Enter your move (rock, paper, or scissors).

3. The computer will make its choice.

4. The result will be displayed: You win, Computer wins, or It's a tie.

5. To stop playing, type quit or n when asked to play again.
