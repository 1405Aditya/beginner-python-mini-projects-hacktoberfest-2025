# ASCII Art Generator from Image

This tool generates images using ASCII characters from an image.

## Description

An ASCII image/art generation tool that uses the Python Image Library to read images and generate its ASCII equivalent using the characters ' _.,-=+:;cba!?0123456789$W#@Ã‘'.

## How to Run?

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies:
```Bash
pip3 -r requirements.txt
```
4. Run the command:
```Bash
python3 main.py
```
5. Specify the path of the target image
